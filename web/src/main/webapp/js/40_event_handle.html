자바스크립트 이벤트 처리

이벤트(Event) : 태그(Element 객체)에서 발생될 수 있는 다양한 사건

이벤트 처리(Event Handle) : 태그에서 이벤트가 발생될 경우 실행될 명령을 자바스크립트로 작성

이벤트 처리 방법-1 : 이벤트가 발생될 태그의 이벤트 리스너 속성(onXXX)에 속성값으로 이벤트 처리 명령을 작성하여 실행 - 이벤트 처리 명령으로 이벤트 처리 함수를 호출하는 것을 권장

이벤트 처리 방법-2 : DOM 트리에서 Element 객체를 찾아 Element 객체의 이벤트 리스너 속성에 이벤트 처리 함수 저장 - 태그에 이벤트 처리 함수 등록

이벤트 처리 방법-3 : DOM 트리에서 Element 객체를 찾아 Element 객체로 addEventLister() 멤버함수를 호출하여 이벤트에 대한 이벤트 처리 함수 등록
