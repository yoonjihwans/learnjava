자바스크립트 객체

자바스크립트는 객체 지향 프로그래밍 언어로 객체를 사용해 프로그램 작성 - 자바스크립트는 모든 값을 객체로 표현하여 처리

객체(Object) : 현실에 존재하는 특정 대상을 프로그램에서 사용하기 위해 속성(Property)과 함수(Function)으로 구성하여 표현한 값

자바스크립트는 클래스(Class) 자료형 대신 생성자 함수(Prototype)를 사용해 객체 생성 - new 연산자로 생성자 함수를 호출하여 객체 생성

생성자 함수를 사용하여 객체를 생성하는 이유는 동일한 속성과 함수를 가지고 있는 다수의 객체를 쉽게 생성하기 위해 사용 - 생산성 및 유지보수의 효율성 증가
