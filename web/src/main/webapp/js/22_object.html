Object 생성자 함수

브라우저에서 객체를 생성할 수 있는 생성자 함수가 미리 선언되어 사용 가능

Object 생성자 함수 : Object 객체를 생성하기 위한 생성자 함수 - 모든 생성자 함수는 Object 생성자 함수를 상속받아 사용 가능

Object 객체 : 객체의 속성이 하나도 없는 객체 - toString 객체 함수만 존재

Object.toString() : 객체를 생성한 생성자 함수의 이름을 반환하는 객체함수 - 객체형 변수를 출력할 경우 자동 호출되는 메소드

Object 객체에 필요한 속성과 함수를 추가하여 사용 가능
