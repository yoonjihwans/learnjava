DOM(Document Object Model)

DOM : Document 객체를 기반으로 제공되는 자바스크립트 객체 - Node 객체

클라이언트는 브라우저를 사용하여 서버의 웹프로그램을 요청해 실행결과를 HTML 문서로 응답받아 브라우저 엔진의 DOM 파서(Parser)로 HTML 문서를 읽어 DOM 트리(Tree)를 생성한 후 CSS 스타일 시트를 적용해 브라우저에 출력 처리

자바스크립트를 사용해 태그에서 이벤트가 발생될 경우 DOM 트리에 존재하는 태그(Node 객체)를 찾아 태그을 조작해 동적인 페이지 구현 가능 - DHTML(Dynamic HTML)

DOM 트리가 완성돼야 동적인 페이지 구현 가능

Node 객체 : 태그(TagNode)와 태그내용(TextNode)을 자바스크립트 객체로 표현 - Element 객체와 Text 객체로 분리하여 표현 가능

Element 객체 : 태그(TagNode)만을 자바스크립트 객체로 표현

Text 객체 : 태그내용(TextNode)만을 자바스크립트 객체로 표현
