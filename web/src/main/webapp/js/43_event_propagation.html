자바스크립트 이벤트 처리

이벤트 전달(Event Propagation) : 브라우저에서 발생된 이벤트는 window 객체에 먼저 도착한 후 DOM 트리에서 이벤트가 발생된 Element 객체(태그)로 이벤트가 이동하여 이벤트 처리 후 반대 방향으로 이동해 window 객체로 되돌아와 이벤트 소멸

이벤트가 이동되는 과정은 캡쳐링 단계와 버블링 단계로 구분

캡쳐링(Capturing) : window 객체로부터 이벤트가 발생된 태그까지의 모든 태그에 이벤트가 전달되는 과정 - 이벤트가 전달된 태그에 등록된 이벤트 처리 함수 호출

버블링(bubbling) : 이벤트가 발생된 태그에서 window 객체까지의 모든 태그에 이벤트가 전달되는 과정 - 이벤트가 전달된 태그에 등록된 이벤트 함수 호출(기본)
